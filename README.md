# Currículo de Engenharia de Software
Base conceitual: SE2014 (SEEK), CC2020 (Competency Model)
Carga total: ~3.400 horas | Duração: 8 semestres | Total de disciplinas: 44

## 1. Filosofia Central
Este currículo não forma "programadores". Forma engenheiros de software capazes de projetar, construir, operar, avaliar e evoluir sistemas de software críticos, considerando custo, risco, qualidade, infraestrutura, pessoas e impacto organizacional.
O currículo segue o modelo de competência (CC2020):
Conhecimento (K): fundamentos teóricos rigorosos
Habilidade (S): aplicação prática mensurável
Disposição (D): postura profissional, ética, rigor técnico
Cada disciplina:
possui objetivo de engenharia explícito
produz artefatos concretos
ocupa um papel claro na linha de evolução cognitiva do estudante

## 2. Linha de Evolução do Estudante
Ano 1 — Pensa em programas
- Lógica, código correto, abstrações básicas, rigor matemático.

Ano 2 — Pensa em sistemas
- Múltiplos componentes, dados, concorrência, requisitos.

Ano 3 — Pensa em organizações e infraestrutura
- Arquitetura, escala, confiabilidade, operação, custo.

Ano 4 — Pensa em impacto, risco e decisão
- Engenharia econômica, governança, sistemas reais, falhas reais.
